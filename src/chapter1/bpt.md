## Task 4

### 4.1 : Installing bpt

Installing bpt is relatively simple. Simply run.

```sh
# Linux
curl bpt.pizza/get/linux -Lo bpt

# MacOS
curl bpt.pizza/get/macos -Lo bpt

# Both
chmod a+x bpt
./bpt install-yourself
```
